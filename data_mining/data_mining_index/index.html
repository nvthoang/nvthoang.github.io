Data Mining

For You

Hey! I hope you enjoy applying data mining skills and techniques to discover hidden patterns and insights from data. I write these blogs as a guideline for beginners as well as for myself to recall knowledge. If you're an expert, these blogs may not be for you.

What You Need

When I write my blogs, I assume that you have basic statistics knowledge and Python programming skills. You may need to download and install Anaconda (which includes Jupyter Notebook) or use Google Colab for cloud-based analysis.

What to Expect

The blogs contain as little theory as possible with plain and simple explanations to get you straight to the point. You are expected to know how to code and interpret results after each topic.
