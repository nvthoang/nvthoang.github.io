Data Analysis

For You

Hey! I hope you enjoy applying data analysis skills and techniques to discover data stories. I write these blogs as a guideline for beginners as well as for myself to recall knowledge. If you're an expert, these blogs may not be for you.

What You Need

When I write my blogs, I assume that you have basic statistics knowledge and R-programming skills. You may need to download and install RStudio.

What to Expect

The blogs contain as little theory as possible with plain and simple explanations to get you straight to the point. You are expected to know how to code and interpret results after each topic.
